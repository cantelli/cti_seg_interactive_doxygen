Detailed Description

A class defining the representation for the vtkGRBFWidget.

This class is a concrete representation for the vtkGRBFWidget. It represents a sphere. Through interaction with the widget, the sphere can be arbitrarily positioned and scaled in 3D space; Typically the vtkGRBFWidget/vtkGRBFRepresentation are used to position a sphere for the purpose of extracting, cutting or clipping data;

To use this representation, you normally use the PlaceWidget() method to position the widget at a specified region in space. It is also possible to set the center of the sphere and a radius.

vtkGRBFWidget
