Detailed Description

A class to propagate a selection of voxels inside a vtkImageData.

This class fills a vtkImageData voxel space propagating a distance from the selection (sphere) center into the voxel volume (tissue). The distance between two voxels is computed mixing the euclidean distance AND the difference of intensity between the voxel and the reference value. The reference value is set manually. If the tolerance is 0, the propagation between two adjacent voxels happens if the value of the next voxel is higher than the threshold (continuous threshold). It tolerance is > 0, the propagation between two adjacent voxels happens if the value of the next voxel is between (1 - tolerance) and (1 + tolerance) times the previous voxel's value.

Warning
The class does NOT allocate (nor delete) the output vtkImageData.
