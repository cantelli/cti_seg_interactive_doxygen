Detailed Description

A class to write Wavefront OBJ (ASCII) files.

vtkOBJWriter - Write a Wavefront OBJ (ASCII) file. The core of this class was taken from vtkOBJExporter. The goal is to make it follow the standard "writer" syntax that is present for the rest of the VTK writers (vtkPLYWriter, etc). It only saves the geometry (vertices, lines, polylines, strips) with normals and tcoords (if any). The faces winding can be flipped by setting FlipVertexOrder to true (ABC => ACB winding).

Warning
It does not write triangle strips properly.
