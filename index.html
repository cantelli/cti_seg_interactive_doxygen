HEIG-VD Interactive Segmentation Tool for Symbios BoneSurfacer

Introduction

This tool is the result of a CTI project developed by HEIG-VD Yverdon.
The code is tailored on measure to Symbios BoneSurfacer tool and fulfills Symbios needs in terms of 3D interactive segmentation of the knee bones/articulation.

The project includes:

  • a VTK filter computing fast implicit modeling, turning a triangulated mesh into a voxelspace map where each voxel contains the distance to the mesh surface (WP5).
  • a VTK widget allowing the selection of continuos tissue regions (a.k.a. segmentation) inside the input dataset (WP5, WP6).

Three algorithms are available through the selection widget to propagate the selection inside the voxelspace (WP6), in all three cases the voxel selection is recursively propagated from the sphere's center:

  • threshold based (threshold continuous selection): the neighbouring voxels (of a selected voxel) are selected if their scalar value is greater than the threshold. Smaller values "slow down" the propagation.
  • tolerance based (gradient continuous selection): the neighbouring voxels (of a selected voxel) are selected if their scalar value is between I*(1-tolerance) and I*(1+tolerance) where I is the selected voxel intensity. Values outside this bounds "slow down" the propagation.
  • bounding-box threshold based (bound threshold continuous selection): same as threshold based but propagation happens (is constrained) inside a bounding box.

The first algorithm is enabled if selectionMode = 0. It works when the bones are not attached (i.e. there's a gap between different bones). Threshold indicates the value below which voxels are not selected. Tolerance and boundsID parameters are not used.
The second algorithm is enabled if selectionMode = 1. It is meant to be used when the bones are attached (i.e. there's no space inbetween). Tolerance should be > 0.0, boundsID parameter is not used.
The third algorithm is enabled if selectionMode = 2. The selection happens in the same way as if selectionMode = 0, but is limited inside a bounding box (boundsID contains the bounds coordinates is voxelspace units). Tolerance parameter is not used.
